#### Makefile

#### 查看命令

```
make or make help
```
####  初始一键设置 初始化rabbitmq/mysql数据库，vendor刷新

```
make awesome_init
```

#### 启动服务器，前台

```
make sd_up
```

#### 守护进程启动服务器

```
make sd_up_background
```

#### rabbitmq 设置 

```
make mq_init
```

#### 创建 stracklog 数据库 

```
make db_create
```
